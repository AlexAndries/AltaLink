UNDERSTAND

Thoroughly understand the current state of the
process, identify the waste and validate the problem
or opportunity

  • Collect reliable and relevant facts and data
  • Identify stakeholders
  • Develop a process map of the current state
  • Identify the waste inherent in the process
  • Pinpoint the root causes of the problem
  • Take action to solve any obvious problems that can be fixed immediately
Next

KEY QUESTIONS TO ANSWER

  • Why does the problem occur? What are the barriers to improvement?
  • How can we measure it?
  • What are the steps in this process? Are there any unnecessary steps?

COMMON TOOLS AND TECHNIQUES:

  • Process mapping
  • Waste analysis
  • Value stream mapping
  • Cause and effect diagrams
  • Root cause analysis

MINIMUM DELIVERABLE

  • Reasons why the problem occurs
  • Clear understanding of the current state of the process
